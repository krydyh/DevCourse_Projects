# ad-control1-simulproject-team4
자율주행 제어1기 시뮬레이션 서빙로봇 프로젝트를 위한 팀 레포

## 서빙 미션 진행 순서
* 1번 위치에서 spawn
* 2번 위치에서 5초 대기하여 음식 받아오기
* 3, 4, 5번 순으로 음식을 서빙
* 6번 위치로 손님들이 사용한 컵, 접시 가져다 주기
